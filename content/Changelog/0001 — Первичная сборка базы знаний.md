# 0001 — Первичная сборка базы знаний

**Когда:** ранее (в ходе проекта)

> [!quote] Запрос
> Сделать Obsidian-базу знаний по нашей D&D-кампании из транскрибций (папка `Trans`) —
> для игроков и ГМ. Формат: подробный пересказ сессий + факты, единый волт.

## Изменено
- Создан волт с папками: Персонажи, NPC, Локации, Мир, Боги/Фракции, Существа, Предметы, Сюжет, Сессии.
- Дашборд [[00 Старт]] и служебный [[_Глоссарий]] (канон-имена ↔ алиасы).
- Заметки по сессиям (пролог и 1–7) и ключевым сущностям мира, собранные из транскрибций.
